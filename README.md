# Installation
+ To get started, the following steps needs to be taken:
+ Clone the repo.
+ `cp .env.example .env`
+ `npm install`
+ `npm run dev`
+ `composer install`
+ `php artisan key:generate`
+ `sudo chown -R $USER:www-data storage`
+ `sudo chown -R $USER:www-data bootstrap/cache`
+ `chmod -R 777 storage`
+ `chmod -R 777 bootstrap/cache`
+ start local server
